# cdwilson.dev

Chris Wilson's Personal Website

## Local Development Environment

Install [Hugo](https://gohugo.io/):

```sh
brew install hugo
```

Install [Invoke](https://www.pyinvoke.org/index.html):

```sh
pip install invoke
```

## Local Build & Preview

```sh
inv preview
```
